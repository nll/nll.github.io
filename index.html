Install VNC Server on Ubuntu 13.04

With vnc server we can connect to a linux machine with desktop an environment. This method is not the more secure, but is very easy, and perfect for local networks. In addition allows each user to work in their area and create multiple remote displays.

First install vncserver

sudo apt-get install vnc4server

And then create one vnc display

vncserver

(each time you run this command one new display is created, and in the first time, a password will be asked to restrict access)

In the end will be displayed the display number, in this case :1

New 'hostname:1 (username)' desktop is pc-hostname:1

To connect to this vnc display you can use any vnc client you like, taking into account that the port is according to your display, :1 will be 5901, :2 will be 5902, and so on…

I choose to use gnome fallback to be my vnc desktop environment, and so I install gnome fallback

sudo apt-get install gnome-session-fallback

And edit the file ~/.vnc/xstartup to this

And finally to stop one vnc display server

vncserver -kill :1
